## ATM Simulator System 
A robust Java-based console application that simulates real-world ATM functionalities. This project demonstrates core Object-Oriented Programming (OOP) principles, including encapsulation, inheritance, and abstraction, to manage user accounts and secure financial transactions.

# Features
* Secure Login: User authentication via Card Number and PIN.
* Balance Inquiry: Check current account balance in real-time.
* Cash Withdrawal: Withdraw funds with automated balance updates and error handling for insufficient funds.
* Deposits: Seamlessly add funds to the account.
* Mini statements: View a log of recent account activity.
* Input Validation: Built-in checks to ensure users enter valid numbers and selections.

# Technologies Used
* Language: Java (JDK 11 or higher recommended)
* IDE: IntelliJ IDEA
* Database: MySql

# How to Run
1. **Clone the repository**
2. **Open in IntelliJ:**
* Launch IntelliJ IDEA.
* Click File > Open.
* Select the cloned project folder.

3. **Create Sql database for operation**
* Create an sql database for data storing of this project.
* Keep name of database and database columns same as used in project.(Just as given in "Database for ATM-Simulator file".)
  
5. **Run the Application:**
* Locate the login file.
* Right-click the file and select Run.
* Interact: Interact with ATM like system.
